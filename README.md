# KURDISH PLAYER

A powerful and premium IPTV player designed for the Kurdish community.

## Features

- **Multi-Source Support**: Add M3U playlists via URL, File, or Xtream API.
- **Kurdish Localization**: Full support for Kurdish Sorani (ckb) by default.
- **Modern UI**: Kurdish-inspired premium design with Red, Yellow, and Green accents.
- **Performance**: High-speed loading and smooth playback with hardware acceleration.
- **LTR Layout**: Optimized Left-To-Right layout for a consistent modern experience.

## Getting Started

1. Launch the app.
2. Select your source on the onboarding screen.
3. Enjoy your favorite channels!
